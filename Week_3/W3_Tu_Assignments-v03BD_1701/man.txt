

THIS IS THE MANUAL  MOUSAWI DATABASE SYSTEM
YOU WILL FIND HERE THE SYNTAX YOU NEED AS FOLLOWS:
--------------------------------------------------------------
TO CREATE A DATABASE:
--------------------------------------------------------------

	---> CREATE ,DATABASE,"database name"

--------------------------------------------------------------
TO CREATE A TABLE:
--------------------------------------------------------------
	
	---> CREATE ,TABLE, "table name"

--------------------------------------------------------------
TO SPECIFY A DATABASE:
--------------------------------------------------------------

	---> SPECIFY, DATABASE, "database name"

--------------------------------------------------------------
TO SPECIFY A TABLE:
--------------------------------------------------------------

	---> SPECIFY, TABLE, "table name"

--------------------------------------------------------------
TO GET ALL RECORDS FROM TABLE:
--------------------------------------------------------------

	---> GET

--------------------------------------------------------------
TO GET A RECORD FOR A SPECIFIC ID:
--------------------------------------------------------------

	---> GET,"id"

--------------------------------------------------------------
TO GET ALL RECORD WITH ATTRIBUTE NAME:
--------------------------------------------------------------

	---> GET, "attribute name"

--------------------------------------------------------------
TO GET SPECIFIC RECORD WITH CONDITION ON ATTRIBUTE NAME:
--------------------------------------------------------------

	---> GET, "column name", "record"

--------------------------------------------------------------
TO ADD A RECORD:
--------------------------------------------------------------

	---> ADD,"attrb1","attrb2"

--------------------------------------------------------------
TO DELETE A RECORD (BY PRIMARY KEY):
--------------------------------------------------------------

	---> DELETE,ROW,"id"

--------------------------------------------------------------
TO DELETE A TABLE:
--------------------------------------------------------------

	---> DROP,TABLE,"table name"

--------------------------------------------------------------
TO DELTE A DATABASE:
--------------------------------------------------------------

	---> DELETE, DATABASE, "database name"

-------------------------------------------------------------- 
TO ROLLBACK TO LAST SAVED POIN:
--------------------------------------------------------------

	---> ROLLBACK

--------------------------------------------------------------
TO SAVE YOUR WORK:
--------------------------------------------------------------

	---> COMMIT

--------------------------------------------------------------
TO QUIT:
--------------------------------------------------------------

	---> quit or QUIT
--------------------------------------------------------------
